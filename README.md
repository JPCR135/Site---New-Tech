### 🚀 Site New Tech
Bem-vindo ao repositório oficial do site da New Tech! Este projeto é um site desenvolvido para apresentar as soluções e tecnologias da empresa.

# 🛠️ Tecnologias Utilizadas
HTML5: Estrutura semântica do site.

CSS3: Estilização, design responsivo e layout.

Git/GitHub: Controle de versão e hospedagem.

## 📋 Funcionalidades
Design moderno e focado na experiência do usuário.

Responsivo (funciona bem em computadores e celulares).

Código versionado e organizado.
